API Reference

Store Data

POST /api/v1/storage/store
curl -X POST https://api.parvyom.com/v1/storage/store \ -H "Content-Type: application/json" \ -H "Authorization: Bearer $STORAGE_TOKEN" \ -d '{ "data": "base64_encoded_data", "type": "immutable", "replication_factor": 3, "encryption": true }'

Retrieve Data

GET /api/v1/storage/retrieve/{hash}
curl -X GET https://api.parvyom.com/v1/storage/retrieve/QmXYZ123... \ -H "Authorization: Bearer $STORAGE_TOKEN" \ -H "Accept: application/octet-stream"

CueDB Query

POST /api/v1/storage/cuedb/query
curl -X POST https://api.parvyom.com/v1/storage/cuedb/query \ -H "Content-Type: application/json" \ -H "Authorization: Bearer $STORAGE_TOKEN" \ -d '{ "query": "SELECT * FROM documents WHERE type = \"agreement\"", "consistency": "strong", "timeout": 5000 }'

CLI Commands

Storage Operations

# Store file with CueDB coordination parvyom storage store --file document.pdf --type immutable --replicas 3 # Retrieve data by content hash parvyom storage get --hash QmXYZ123... --output retrieved_file.pdf # Query CueDB for coordinated data parvyom storage query --cue "type: agreement" --format json

Cluster Management

# Add storage node to cluster parvyom storage add-node --endpoint https://node.example.com --capacity 10TB # Check cluster health and replication status parvyom storage health --detailed --check-replicas # Repair data integrity across cluster parvyom storage repair --verify-checksums --fix-replicas

Integration Examples

Rust Integration

use parvyom_storage::StorageClient; let storage = StorageClient::new(); // Store data with CueDB coordination let hash = storage.store_data( data.as_bytes(), StorageType::Immutable, ReplicationFactor::Three, EncryptionLevel::AES256 ).await?; // Query CueDB for coordinated access let results = storage.cuedb_query( "SELECT * FROM agreements WHERE status = 'active'", ConsistencyLevel::Strong ).await?;

JavaScript Integration

const { StorageClient } = require('@parvyom/storage'); const storage = new StorageClient(); // Store document with metadata const hash = await storage.store({ data: fileBuffer, type: 'immutable', metadata: { title: 'Agreement', version: '1.0' }, replicas: 3 }); // Retrieve with CueDB coordination const document = await storage.retrieve(hash, { consistency: 'strong', decrypt: true });
