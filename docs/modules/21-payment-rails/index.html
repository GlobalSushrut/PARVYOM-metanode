Payment Rails System

Multi-rail payment system supporting ACH, SEPA, INTERAC, RTP, and cross-border settlements with real-time processing

System Overview

The Payment Rails System provides comprehensive support for multiple payment networks, enabling seamless domestic and international transactions with real-time settlement capabilities.

Supported Payment Rails

  • ACH (Automated Clearing House)
  • SEPA (Single Euro Payments Area)
  • INTERAC (Canada)
  • RTP (Real-Time Payments)
  • FedNow (US Federal Reserve)
  • Cross-border SWIFT

API Reference

Initiate Payment

POST /api/v1/payments/initiate
curl -X POST https://api.parvyom.com/v1/payments/initiate \ -H "Content-Type: application/json" \ -H "Authorization: Bearer $PAYMENT_TOKEN" \ -d '{ "amount": 1000.00, "currency": "USD", "rail": "RTP", "sender_account": "123456789", "receiver_account": "987654321", "settlement_speed": "instant" }'

Cross-Border Transfer

POST /api/v1/payments/cross-border
curl -X POST https://api.parvyom.com/v1/payments/cross-border \ -H "Content-Type: application/json" \ -H "Authorization: Bearer $PAYMENT_TOKEN" \ -d '{ "amount": 5000.00, "source_currency": "USD", "target_currency": "EUR", "source_rail": "ACH", "target_rail": "SEPA", "compliance_check": true }'

Payment Status

GET /api/v1/payments/status/{payment_id}
curl -X GET https://api.parvyom.com/v1/payments/status/pay_123456 \ -H "Authorization: Bearer $PAYMENT_TOKEN"

CLI Commands

Payment Operations

# Send payment via specific rail parvyom payment send --amount 1000 --currency USD --rail RTP --to-account 123456789 # Check payment status parvyom payment status --payment-id pay_123456 --detailed # List supported rails for region parvyom payment rails --region US --capabilities instant

Settlement Management

# Configure settlement preferences parvyom payment configure --rail ACH --settlement-speed same-day --retry-policy 3 # Monitor cross-border transfers parvyom payment monitor --cross-border --real-time --alerts # Generate settlement report parvyom payment report --period monthly --format pdf --include-fees

Integration Examples

Rust Integration

use parvyom_payments::PaymentRails; let payment_system = PaymentRails::new(); // Send payment via RTP let payment = payment_system.send_payment( PaymentRequest { amount: 1000.00, currency: Currency::USD, rail: PaymentRail::RTP, sender: "123456789".to_string(), receiver: "987654321".to_string(), settlement_speed: SettlementSpeed::Instant, } ).await?; // Cross-border transfer let transfer = payment_system.cross_border_transfer( 1000.00, Currency::USD, Currency::EUR, PaymentRail::ACH, PaymentRail::SEPA ).await?;

JavaScript Integration

const { PaymentRails } = require('@parvyom/payments'); const payments = new PaymentRails(); // Initiate payment const payment = await payments.initiatePayment({ amount: 1000.00, currency: 'USD', rail: 'RTP', senderAccount: '123456789', receiverAccount: '987654321', settlementSpeed: 'instant' }); // Monitor payment status const status = await payments.getPaymentStatus(payment.id); console.log('Payment status:', status.state);
