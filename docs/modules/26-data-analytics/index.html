API Reference

Data Query

POST /api/v1/analytics/query
curl -X POST https://api.parvyom.com/v1/analytics/query \ -H "Content-Type: application/json" \ -H "Authorization: Bearer $ANALYTICS_TOKEN" \ -d '{ "dataset": "transactions", "query": "SELECT * FROM transactions WHERE amount > 1000", "timeframe": "last_30_days", "aggregation": "sum" }'

Machine Learning Model

POST /api/v1/analytics/ml/predict
curl -X POST https://api.parvyom.com/v1/analytics/ml/predict \ -H "Content-Type: application/json" \ -H "Authorization: Bearer $ANALYTICS_TOKEN" \ -d '{ "model": "fraud_detection", "features": { "amount": 5000, "merchant": "online_store", "time": "2024-01-15T10:30:00Z" } }'

CLI Commands

Data Operations

# Query data with analytics parvyom analytics query --dataset transactions --sql "SELECT * FROM sales" --format json # Train ML model parvyom analytics ml train --model fraud_detection --dataset training_data.csv # Generate report parvyom analytics report --template monthly_sales --output report.pdf

Dashboard Management

# Create dashboard parvyom analytics dashboard create --name "Sales" --template sales_template # Export dashboard data parvyom analytics dashboard export --id dash_123 --format csv # Schedule automated report parvyom analytics schedule --report monthly --frequency weekly --email team@company.com

Integration Examples

Rust Integration

use parvyom_analytics::AnalyticsClient; let analytics = AnalyticsClient::new(); // Query data let results = analytics.query_data( "transactions", "SELECT * FROM transactions WHERE amount > 1000", QueryOptions { timeframe: "last_30_days".to_string(), limit: Some(1000), } ).await?; // ML prediction let prediction = analytics.predict( "fraud_detection", vec![ ("amount", 5000.0), ("merchant", "online_store".to_string()), ] ).await?;

JavaScript Integration

const { AnalyticsClient } = require('@parvyom/analytics'); const analytics = new AnalyticsClient(); // Create dashboard const dashboard = await analytics.createDashboard({ name: 'Sales Dashboard', widgets: [ { type: 'chart', dataSource: 'sales_data', chartType: 'line' } ] }); // Real-time data streaming analytics.streamData('transactions', (data) => { console.log('New transaction:', data); });
