API Reference

Identity Creation

POST /api/v1/identity/create
curl -X POST https://api.parvyom.com/v1/identity/create \ -H "Content-Type: application/json" \ -H "Authorization: Bearer $IDENTITY_TOKEN" \ -d '{ "identity_type": "individual", "biometric_data": "encrypted_biometric_hash", "public_key": "ed25519_public_key", "privacy_level": "maximum" }'

Zero-Knowledge Verification

POST /api/v1/identity/verify-zk
curl -X POST https://api.parvyom.com/v1/identity/verify-zk \ -H "Content-Type: application/json" \ -H "Authorization: Bearer $VERIFIER_TOKEN" \ -d '{ "identity_id": "did:parvyom:123456789", "proof_request": "age_over_18", "challenge": "random_challenge_string" }'

CLI Commands

Identity Management

# Create new digital identity parvyom identity create --type individual --biometric fingerprint --privacy-level max # Generate zero-knowledge proof parvyom identity prove --claim age_over_18 --identity-id did:parvyom:123456789 # Verify identity with biometrics parvyom identity verify --biometric fingerprint --challenge-response

Privacy Operations

# Create selective disclosure proof parvyom privacy selective-disclosure --attributes name,age --hide-others # Generate anonymous credential parvyom privacy anonymous-credential --issuer government --claim citizenship # Verify without revealing identity parvyom privacy verify-anonymous --proof zk-proof.json --requirements age_over_21

Rust Integration

const { DigitalIdentity } = require('@parvyom/identity'); const identity = new DigitalIdentity(); // Create identity const did = await identity.createIdentity({ type: 'individual', biometric: fingerprintHash, privacyLevel: 'maximum' }); // Verify with zero-knowledge const isValid = await identity.verifyZK({ did: did, claim: 'age_over_18', proof: zkProof });
