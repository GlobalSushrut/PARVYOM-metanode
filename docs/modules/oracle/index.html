API Reference

Core Oracle Operations

GET
/api/oracle/health
Get oracle system health and status
POST
/api/oracle/nodes/register
Register new BPI node with oracle system
GET
/api/oracle/nodes
List all registered BPI nodes

Cross-System Communication

POST
/api/oracle/cross-system/message
Send cross-system message with routing
GET
/api/oracle/cross-system/status/{request_id}
Get cross-system request status

Data Queries

POST
/api/oracle/query
Execute data query across connected systems
GET
/api/oracle/query/results/{query_id}
Get query results and status

Event Subscriptions

/api/oracle/events/subscribe
Subscribe to real-time events with filtering
DELETE
/api/oracle/events/unsubscribe/{subscription_id}

/api/oracle/partners/register
