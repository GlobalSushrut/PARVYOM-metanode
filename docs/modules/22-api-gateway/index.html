API Reference

Gateway Configuration

POST /api/v1/gateway/configure
curl -X POST https://api.parvyom.com/v1/gateway/configure \ -H "Content-Type: application/json" \ -H "Authorization: Bearer $GATEWAY_TOKEN" \ -d '{ "routes": [ { "path": "/api/v1/payments/*", "upstream": "https://payments.internal:8080", "rate_limit": 1000, "auth_required": true } ] }'

Rate Limiting

POST /api/v1/gateway/rate-limit
curl -X POST https://api.parvyom.com/v1/gateway/rate-limit \ -H "Content-Type: application/json" \ -H "Authorization: Bearer $GATEWAY_TOKEN" \ -d '{ "api_key": "key_123456", "requests_per_minute": 1000, "burst_limit": 100, "quota_reset": "daily" }'

Analytics

GET /api/v1/gateway/analytics
curl -X GET https://api.parvyom.com/v1/gateway/analytics \ -H "Authorization: Bearer $GATEWAY_TOKEN" \ -G -d "timeframe=24h" -d "metrics=requests,latency,errors"

Gateway Management

# Deploy gateway configuration parvyom gateway deploy --config gateway.yaml --environment production # Monitor gateway performance parvyom gateway monitor --real-time --metrics requests,latency,errors # Update rate limits parvyom gateway rate-limit --api-key key_123 --limit 1000/min --burst 100

Route Management

# Add new API route parvyom gateway route add --path "/api/v2/*" --upstream https://api-v2.internal # Test route configuration parvyom gateway route test --path "/api/v1/test" --method GET # Export gateway metrics parvyom gateway export --format json --timeframe 24h --output metrics.json

Integration Examples

Rust Integration

use parvyom_gateway::GatewayClient; let gateway = GatewayClient::new(); // Configure API route let route = gateway.add_route( "/api/v1/users/*", "https://users.internal:8080", RouteConfig { rate_limit: 1000, auth_required: true, timeout: Duration::from_secs(30), } ).await?; // Monitor gateway metrics let metrics = gateway.get_metrics( MetricsQuery { timeframe: "1h".to_string(), include: vec!["requests", "latency", "errors"], } ).await?;

JavaScript Integration

const { GatewayClient } = require('@parvyom/gateway'); const gateway = new GatewayClient(); // Configure rate limiting await gateway.setRateLimit({ apiKey: 'key_123456', requestsPerMinute: 1000, burstLimit: 100, quotaReset: 'daily' }); // Get real-time analytics const analytics = await gateway.getAnalytics({ timeframe: '24h', metrics: ['requests', 'latency', 'errors'] });
