API Reference

Metrics Collection

POST /api/v1/monitoring/metrics
curl -X POST https://api.parvyom.com/v1/monitoring/metrics \ -H "Content-Type: application/json" \ -H "Authorization: Bearer $MONITORING_TOKEN" \ -d '{ "metric_name": "cpu_usage", "value": 75.5, "timestamp": "2024-01-15T10:30:00Z", "labels": { "host": "server-01", "environment": "production" } }'

Alert Configuration

POST /api/v1/monitoring/alerts
curl -X POST https://api.parvyom.com/v1/monitoring/alerts \ -H "Content-Type: application/json" \ -H "Authorization: Bearer $MONITORING_TOKEN" \ -d '{ "name": "High CPU Usage", "condition": "cpu_usage > 80", "severity": "warning", "notification_channels": ["email", "slack"] }'

Dashboard Query

GET /api/v1/monitoring/dashboard
curl -X GET https://api.parvyom.com/v1/monitoring/dashboard \ -H "Authorization: Bearer $MONITORING_TOKEN" \ -G -d "timeframe=1h" -d "metrics=cpu,memory,disk"

CLI Commands

Monitoring Operations

# Start monitoring agent parvyom monitor start --config monitoring.yaml --metrics-port 9090 # Query metrics parvyom monitor query --metric cpu_usage --timeframe 1h --host server-01 # Set up alert parvyom monitor alert create --name "High Memory" --condition "memory > 90%" --severity critical

Dashboard Management

# Create dashboard parvyom monitor dashboard create --name "System Overview" --template infrastructure # Export metrics data parvyom monitor export --format csv --timeframe 24h --output metrics.csv # Health check parvyom monitor health --all-services --detailed --format json

Integration Examples

Rust Integration

use parvyom_monitoring::MonitoringClient; let monitoring = MonitoringClient::new(); // Send metrics monitoring.send_metric( "api_response_time", 125.5, vec![ ("endpoint", "/api/users"), ("method", "GET"), ] ).await?; // Create alert let alert = monitoring.create_alert( "API Latency Alert", "api_response_time > 1000", AlertSeverity::Warning, vec!["email", "slack"] ).await?;

JavaScript Integration

const { MonitoringClient } = require('@parvyom/monitoring'); const monitoring = new MonitoringClient(); // Track custom metric await monitoring.trackMetric({ name: 'user_signups', value: 1, labels: { source: 'web', campaign: 'summer2024' } }); // Query dashboard data const dashboardData = await monitoring.getDashboard({ timeframe: '1h', metrics: ['cpu', 'memory', 'requests'] });
