System Overview

The BPI Oracle System provides a decentralized network of oracle nodes that securely bridge real-world data with blockchain applications, enabling smart contracts to access external information with cryptographic guarantees.

API Reference

Data Request

POST /api/v1/oracle/request
curl -X POST https://api.parvyom.com/v1/oracle/request \ -H "Content-Type: application/json" \ -H "Authorization: Bearer $ORACLE_TOKEN" \ -d '{ "data_source": "financial_markets", "query": "BTC/USD", "aggregation": "median", "min_responses": 5, "max_deviation": 0.01 }'

Cross-Chain Communication

POST /api/v1/oracle/cross-chain
curl -X POST https://api.parvyom.com/v1/oracle/cross-chain \ -H "Content-Type: application/json" \ -H "Authorization: Bearer $ORACLE_TOKEN" \ -d '{ "source_chain": "ethereum", "target_chain": "bpi", "data_type": "token_price", "verification_required": true }'

CLI Commands

Oracle Operations

# Start oracle node parvyom oracle start --node-id oracle_001 --stake 10000 # Request data from oracle network parvyom oracle request --source financial --query "ETH/USD" --min-responses 5 # Register as oracle provider parvyom oracle register --data-sources financial,weather,iot --public-key $PUB_KEY

Cross-Chain Operations

# Setup cross-chain bridge parvyom oracle bridge --source ethereum --target bpi --verify-proofs # Monitor oracle network health parvyom oracle monitor --metrics accuracy,latency,availability # Aggregate data from multiple sources parvyom oracle aggregate --sources api1,api2,api3 --method median

Integration Examples

Rust Integration

use parvyom_oracle::OracleClient; let oracle = OracleClient::new(); // Request price data let price_data = oracle.request_data( DataSource::Financial, "BTC/USD", AggregationMethod::Median, 5 // min responses ).await?; // Cross-chain data transfer let result = oracle.cross_chain_transfer( "ethereum", "bpi", TokenPriceData::new("ETH", price_data.price) ).await?;

JavaScript Integration

const { OracleClient } = require('@parvyom/oracle'); const oracle = new OracleClient(); // Request weather data const weatherData = await oracle.requestData({ source: 'weather', query: 'temperature_NYC', aggregation: 'average', minResponses: 3 }); // Subscribe to real-time updates oracle.subscribe('BTC/USD', (priceUpdate) => { console.log('New price:', priceUpdate.price); });
