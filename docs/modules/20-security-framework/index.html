CLI Commands

Security Operations

# Generate post-quantum keypair parvyom security keygen --algorithm dilithium5 --output pq_keypair.pem # Encrypt data with post-quantum security parvyom security encrypt --file sensitive.txt --algorithm pq-aes --key pq_key.pem # Verify cryptographic signatures parvyom security verify --signature sig.bin --public-key pub.pem --message msg.txt

Threat Management

# Start threat monitoring parvyom security monitor --real-time --threat-model advanced # Analyze security events parvyom security analyze --events security.log --ml-detection # Generate security report parvyom security report --compliance military --format pdf

Integration Examples

Rust Integration

use parvyom_security::SecurityFramework; let security = SecurityFramework::new(); // Post-quantum encryption let encrypted = security.encrypt_post_quantum( sensitive_data.as_bytes(), &post_quantum_key, EncryptionAlgorithm::PostQuantumAES ).await?; // Zero-trust verification let is_authorized = security.verify_zero_trust( &identity_proof, &access_request, TrustLevel::Military ).await?;
