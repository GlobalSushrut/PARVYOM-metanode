API Reference

Send Message

POST /api/v1/messages/send
curl -X POST https://api.parvyom.com/v1/messages/send \ -H "Content-Type: application/json" \ -H "Authorization: Bearer $MESSAGE_TOKEN" \ -d '{ "recipient": "user@example.com", "content": "Hello, this is a secure message", "type": "text", "encryption": "end_to_end" }'

Create Group

POST /api/v1/messages/groups
curl -X POST https://api.parvyom.com/v1/messages/groups \ -H "Content-Type: application/json" \ -H "Authorization: Bearer $MESSAGE_TOKEN" \ -d '{ "name": "Project Team", "members": ["user1@example.com", "user2@example.com"], "privacy": "private", "encryption": true }'

Get Messages

GET /api/v1/messages/history
curl -X GET https://api.parvyom.com/v1/messages/history \ -H "Authorization: Bearer $MESSAGE_TOKEN" \ -G -d "conversation_id=conv_123" -d "limit=50" -d "offset=0"

CLI Commands

Messaging Operations

# Send encrypted message parvyom message send --to user@example.com --text "Hello" --encrypt # Create secure group parvyom message group create --name "Team" --members user1,user2 --private # View message history parvyom message history --conversation conv_123 --limit 50

Privacy Controls

# Set privacy preferences parvyom message privacy --read-receipts off --typing-indicators off # Export message data parvyom message export --format json --timeframe 30d --encrypted # Delete conversation parvyom message delete --conversation conv_123 --permanent

Integration Examples

Rust Integration

use parvyom_messaging::MessagingClient; let messaging = MessagingClient::new(); // Send encrypted message let message = messaging.send_message( "user@example.com", "Hello, this is secure!", MessageType::Text, EncryptionLevel::EndToEnd ).await?; // Create group conversation let group = messaging.create_group( "Project Team", vec!["user1@example.com", "user2@example.com"], GroupPrivacy::Private ).await?;

JavaScript Integration

const { MessagingClient } = require('@parvyom/messaging'); const messaging = new MessagingClient(); // Send message const message = await messaging.sendMessage({ recipient: 'user@example.com', content: 'Hello, secure message!', type: 'text', encryption: 'end_to_end' }); // Listen for real-time messages messaging.onMessage((message) => { console.log('New message:', message.content); });
